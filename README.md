# DKTE Placement Predictor (83.3% Accuracy)

Simple ML project to predict student placement chances based on CGPA, internships, and coding score.

## How it works
- Dataset: 30 synthetic DKTE-style student records (`placement_data.csv`)
- Model: RandomForestClassifier (scikit-learn)
- Accuracy: 83.3% on test split
- Output: Saved model in `placement_model.pkl` for future API integration
